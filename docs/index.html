Try a preset, then watch it converge. Gravistimulate to redistribute auxin to the lower side (the start of root bending); 🛰️ Spaceflight (µG) lowers columella PIN3/7 — as seen in the CARA / OSD-120 ISS root transcriptome — so auxin becomes confined to the QC. PIN2 / aux1 mutant break the return flow / tip trap. The white arrows trace the auxin reflux loop.

Model: two-compartment (cell + apoplast wall) auxin transport with carrier-modulated membrane permeabilities, after Grieneisen et al. 2007 and Band et al. 2014. A teaching model — see the repository and SPEC.md for equations, parameters and limitations. Part of the AIRI astrobotany program. Public domain (CC0).
